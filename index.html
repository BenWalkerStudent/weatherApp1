HTML Project 1

Starting Your Weather and News Site

Purpose

The goal of this project is to build a web site that you build completely on your own. Use some of the skills you've learned from the HTML Cat Photo App project to build a project that will be improved on via CSS in the coming weeks. You will want to use as much knowledge as you have for your project so that you're applying all that you've learned and are able to use that information effectively when creating websites.

Instructions

When done, you will have created an HTML skeleton layout for your weather news site application with the following elements...

  • A header tag with the an image and a nav tag contained inside of it.
  • In the nav tag include menu links to "home", "news" and "weather" as well as a link to an "about" menu item.
  • Article elements with 2 section elements, one for weather, one for news.
  • With in the "Weather" section have a header for "Latest Weather"
  • Have a div element containing a "Weather Summary"
  • In the Weather Summary have the following information
    1. Area the weather is for
    2. Description of the weather
    3. Temperature
    4. Wind Chill
    5. Humidity
    6. Wind Speed
  • Have 5 day forecast in a separate element below the Weather Summary
    1. Include a heading of "5 Day Forecast"
    2. The area the forecast is for
    3. 5 days with the day, a image of what the weather is supposed to be and the temperature.
  • In the section element for the "News" section have a header for "Latest News"
  • Have an image element for a leading news story
  • Below the image element have the news headline with the brief summary of the news story below
  • In a separate div element below the leading story have a place for several cards to display other news stories, each card will need to have a similar set up as the headline news element

Assumptions

    This project assumes:
  1. You have already create the Cat Photo App from the previous assignments.

  3. If you don't understand something, you should feel comfortable asking questions or referring back to the course material.
